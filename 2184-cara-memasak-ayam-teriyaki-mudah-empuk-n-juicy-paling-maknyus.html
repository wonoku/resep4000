Cara Memasak Ayam teriyaki mudah empuk n juicy

  1. Taburkan secukupnya garam k permukaan ayam n tusuk2 dgn garpu.
  2. Campurkan kecap shoyu, gula pasir n madu, aduk sampai tercampur rata.
  3. Masukkan ayam k dlm campuran marinasi, balur hingga merata, diamkan 30 mnt dgn bagian kulit d bwh (sy masukin kulkas semalaman).
  4. Panaskan wajan dgn api kcl, masukkan ayam dgn bagian kulit d bwh, tutup n masak selama 5 mnt, api kecil ya agar tdk gosong.
  5. Setelah 5 mnt..buka tutupnya n balikkan ayam, tutup lg n masak selama 5 mnt msh dgn api kecil.
  6. Setelah 5 mnt matikan api tp jgn dl d buka tutupnya, biarkan dl selama 5 mnt agar ayam matang sempurna.
